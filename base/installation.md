## 安装 Python ##

- 访问 [https://www.python.org/shell/](https://www.python.org/shell/) 可以立即体验 Python 编程。
- 访问 [https://www.python.org/downloads/](https://www.python.org/downloads/) 下载并安装 Python 到电脑中。

### Mac 用户使用 Python ###
Mac 电脑已经预装 Python 。
使用 Python 的步骤：

1. 调出 _Spotlight_
2. 输入 _terminal_
3. 在 _Terminal_ 中输入 _python_ ，回车即可
