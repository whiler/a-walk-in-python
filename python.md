## Python ##
Python 名词，发音 ['paɪθɑn]  ，有两种解释：

1. 一种体型巨大的无毒热带蟒蛇；
2. 一种被广泛使用的解释性动态编程语言。

### 为什么是 Python ###
Python 简单、优雅而且强大。

- **简单**
	相对于 C/C++ 、Java ，Python 简单得令人发指
- **高效**
	Python 是一种解释性语言，没有繁琐的编译、链接过程，开发、调试极其方便
- **可读性强**
	Python 强制用缩进，让逻辑结构清晰明了
- **使用广泛**
	Python 被各个行业使用，使用 Python 的人很多，遇到问题可以方便地得到帮助
